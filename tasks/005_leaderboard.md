# Task 005 --- Leaderboard System

Create command:

/leaderboard

Sort all users by **total net worth**.

Net worth = cash + value of all holdings.

Display top 10 users.

Example:

1.  Alice --- \$63,201
2.  Bob --- \$59,882
3.  Charlie --- \$57,110

## Optimization

Only recompute portfolio values when leaderboard is requested.
